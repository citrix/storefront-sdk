Get-STFInstalledFeatures

Get the details of the Storefront features on the system

Syntax

Detailed Description

Get the details of the Storefront features on the system.

Related Commands

Input Type

Return Values

FeatureClass

Examples

EXAMPLE 1 Get all features installed

Get-STFInstalledFeatures
